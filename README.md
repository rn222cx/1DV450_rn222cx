# 1DV450_rn222cx

### Idé

En applikation där man kan lägga till auktionsfirmor med tillhörande information.
Dessa placeras ut på en karta.
En autensierad användare har CRUD funktionalitet på sina auktionsfirmor och publika användare kan endast läs behörighet.
exempel på taggar: liveauktion, slagauktioner, nätauktioner, konst, smycken
Sortering genom län.
